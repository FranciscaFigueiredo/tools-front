&nbsp;

## How to install and run on terminal


&nbsp;

### - Clone repository
```
  git clone https://github.com/FranciscaFigueiredo/tools-front.git
```

&nbsp;

### - Install dependencies

```
  npm i
```
or
```
  npm install
```

&nbsp;

### - Start

```
  npm start
```
